# Oho语法

Oho lang的所有语句（指令）均为以下形式：
```
<哦xN><齁xN><!xN>
```
例如：
```
哦哦哦齁齁！！！
```
其中，`哦`的数量代表指令id，`齁`的数量代表变量id，`！`的数量代表立即数，所以上面的指令实际上为
```
Inst{3} Var{2} Imm{3}
```

指令之间，哦、齁、！之间可以插入任何数量其他字符，同时支持使用オ、ホ代替哦、齁。

## 变量

未声明变量在被指令使用时会自动被初始化为0，所有变量均为32位，指令按照整型解读变量

以下为一些特殊变量
* Var{1}：用于Load和Store指令
* Var{2}：用于Jump指令

## 内存操作

Oho虚拟机的内存为32位

## 指令

以下指令均按照`Inst Var Imm`解读，例如`哦哦齁！！！`被解读为`Inst{2} Var{1} Imm{3}`，即`Sub: Var{1} = Var{1} - 3`

### Inst{1}: Add

```
Var = Var + Imm
```

### Inst{2}: Sub

```
Var = Var - Imm
```

### Inst{3}: Load

向`Var`写入地址`Var{1} + Imm`的值

### Inst{4}: Store

向地址`Var{1} + Imm`写入`Var`

### Inst{5}: Jump

如果`Var{2} != 0`，向前跳转`Var + Imm`条指令

### Inst{6}: Print

从地址`Var`开始，打印`Imm`个字符，每个地址字符按照32位解释为UTF-8
